Fully Homomorphic Encryption

This project is an implementation of a fully homomorphic encryption system using Python. As this was a learning exercise, we do not focus on efficientcy. This cryptography model allows users to apply arbitrary circuit operations over encrypted data without decrypting it.

Concurrent Elliptic Curve Cryptography

This project developed a parallel Elliptic Curve Cryptography (ECC) module in Erlang. The goal was to develop fast methods in ECC that could be parallelized without general loss of security. I learned Erlang for this project for its friendly concurrency and bit-manipulation language features.

AdaRailz

AdaRailz was a senior project hosted by WWU. Our department build a model train set and had three sets of students contribute to a waterfall development model over three academic quarters. One team did specifications analysis, the second team did design development, and I participated in the implementation team. The project specification was to design a safe and concurrent Model Train control system. Our team implemented human interface tools, systems safety to prevent collisions and derailments, and enabled critical section reservations. This project was written in Ada.

Petrographic Database

I participated in a group project to design a petrographic database, to aid in mineral identification in rock samples, for the WWU Geology department. The project provided an interface that allowed users to screen and score minerals by their measurable attributes. My greatest influence in the project was developing the mineral scoring algorithm. Having never taken a geology course, I found this project enforcing my interdisciplinary communication skills. The foundations that our team laid are now being passed on as a vehicle for graduate research.
